
# Spring-boot Ecommerce Shop project from Hutech Student

## Technology Stack
* FrontEnd:
    - ReactJs
    - Material UI
    - Tailwind CSS
    - Axios
    - Redux
    - Redux Toolkit
* BackEnd:
    - Spring-boot 2.7.0
    - Java 17
    - Spring Data Jpa
    - Spring Security
    - Jwt
    - MySql
* Database: MySql latest version
* Tool build: Docker
* IDEs: Intellij Idea 22.3, Visual Studio Code, Navicat 16.x
